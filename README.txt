README

This project contains code and instructions to localize intracranial electrodes from the pre-op MRI and post-op CT. 
This code requires MATLAB, freesurfer and Slicer 3D to be properly installed.

Usage is governed by: http://klab.tch.harvard.edu/code/license_agreement.pdf

CONTENTS

how_to_coregister_freeview.pdf - instructions about the localization procedure
convertXLStoLabels.m - the core code to convert the localized electrodes to a MAT format and assign region labels. Paths may have to be modified for your use.
m00118.xls - dummy example data
